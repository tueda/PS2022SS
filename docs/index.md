さらに、次のことも知っておくと有益かもしれません。

- よく使われる標準ライブラリおよびサードパーティのライブラリ

- docstringや型ヒントの書き方
- アプリケーションおよびライブラリのパッケージング・単体テスト
- （Pythonに限らず）もっと高度な計算アルゴリズムやデータ構造
